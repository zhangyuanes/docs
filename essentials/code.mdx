## 基础

### 内联代码

要将 `word` 或 `phrase` 表示为代码，请用反引号 (`) 将其括起来。

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### 代码块

使用 [围栏代码块](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)，用三个反引号将代码括起来，并在前导反引号后跟代码片段的编程语言以获得语法高亮。可选地，您也可以在编程语言后写入代码的名称。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

````md
```java HelloWorld.java
````
